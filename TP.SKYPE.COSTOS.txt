¿Qué es Skype?
::::::::::::::

Skype es un software que permite que todo el mundo se comunique. Millones de personas y empresas usan Skype para hacer llamadas y videollamadas gratis individuales y grupales, enviar mensajes instantáneos y compartir archivos con otras personas que usan Skype, mediante teléfono, celular, PC o tableta.


Las funciones gratuita entre usuarios de Skype son: 
- LLAMADAS: Las llamadas entre usuarios de Skype son siempre gratuitas, tanto a lugares cercanos como llamadas internacionales, la duracion es "ilimitada".
- VIDEOLLAMADAS gratuitas, ya sean individuales y grupales. Para acceder a este servicio de comparti una videollamada gratis con cualquier persona, requiere que la persona receptora cuente con Skype y una cámara web o un smartphone compatible, mediante videollamadas de gran calidad.
- ENVIAR MENSAJES INSTANTANEOS: Sin importar dónde se encuentre el emisor, se pueden envíar mensajes sin costo alguno, y desde cualquier lugar. Ademas la aplicación de mensajería instantánea de Skype, posee nuevos emoticonos y Mojis.
-COMPARTIR ARCHIVOS: 
Tambien se pueden compartir archivos pequeños , o  fotos, y otros archivos "pesados" a varios contactos durante una llamada de voz, una videollamada o una llamada grupal,con otras personas que usan Skype. 

De otro modo, SE DEBE ABONAR, los siguientes servicios:  
- LLAMADAS A TELEFONOS FIJOS Y MOVILES: aun sin tener Skype. De este modo las llamadas internacionales resultan, aun cuando la comunicacion sea al otro lado del mundo, como a las cercanias. 
Se requieren crédito de Skype para enviar mensajes de texto.

El pago se puede realizar mediante:

-Crédito de Skype o, 
-un plan para llamar a teléfonos móviles y fijos. 

El precio que se abona, depende del país al que llamas, no del país desde el que se llama.

Otras opciones y diferencias en los usos de Skype, Reuniones de Skype y Skype Empresarial.

* La versión de Skype usual o domestica tambien es perfecta para pequeñas empresas con hasta 20 empleados. El programa es gratis, a menos que se compren crédito para hacer llamadas a fijos y móviles.

* Reuniones de Skype permite contactar de manera gratuita,  con un grupo hasta con 10 personas al mismo tiempo (un máximo de 3 después de 60 días),  en una conferencia web propia en alta definición, desde cualquier dispositivo o sistema operativo. Tambien se pueden pantallas, usar el puntero láser o una pizarra colaborativa. Además, los organizadores tendrán acceso a controles profesionales, como la capacidad de silenciar o reactivar el audio de los participantes. Actualmente, este servicio solo está disponible en Estados Unidos.
* Skype empresarial permite agregar hasta 250 personas a tus reuniones en línea, te ofrece una seguridad de nivel empresarial, que permite administrar las cuentas de empleados y está integrado en tus aplicaciones Office. Cuesta 2 $ al mes por usuario. La videollamada a contactos de Skype Empresarial actualmente solo es compatible con Skype para escritorio de Windows; la videollamada en Skype para Mac estará disponible próximamente.
